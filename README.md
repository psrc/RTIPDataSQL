## RTIPDataSQL

This repo holds the DDL code that defines the structure of the SQL Server database that is the back end to the TIP system.

See the [Database](./Database) folder for the actual DDL code.

See the [Docs](./Docs/SQLserver_documentation-2025-10-20T11-01-15/SQLserver/User_databases/RTIPDataSQL/index.md) for arguably more user-friendly information about each object in the database, including dependencies.
